# 🐍 Python_Basic

> **유튜브 조코딩(JoCoding) 채널의 '점프 투 파이썬' 강의를 기반으로 학습하고 정리한 파이썬 기초 자료형 노트입니다.**
> 데이터 분석가 및 엔지니어로 나아가기 위한 기본 프로그래밍 체력을 다지기 위해 작성되었습니다.

---

## 📌 주요 학습 내용

현재 파이썬의 가장 핵심이 되는 **기본 자료형과 자료구조**의 개념을 이해하고, 주피터 노트북(`.ipynb`) 환경에서 직접 코드를 실행하며 실습을 진행했습니다.

* **자료형의 종류 및 특징 실습**
  * **숫자, 문자열, 불(Boolean):** 데이터의 가장 기본적인 타입 이해
  * **변수, 리스트, 튜플, 딕셔너리, 집합:** 데이터를 효율적으로 담고 관리하는 자료구조의 특징 파악 및 활용

---

* **Language:** Python 3.x
