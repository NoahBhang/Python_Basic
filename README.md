# 🐍 Python_Basic

> **유튜브 조코딩(JoCoding) 채널의 '점프 투 파이썬' 강의를 기반으로 학습하고 정리한 파이썬 기초 자료형 노트입니다.**
> 데이터 분석가 및 엔지니어로 나아가기 위한 기본 프로그래밍 체력을 다지기 위해 작성되었습니다.

---

## 📌 주요 학습 내용

현재 파이썬의 가장 핵심이 되는 **기본 자료형과 자료구조**의 개념을 이해하고, 주피터 노트북(`.ipynb`) 환경에서 직접 코드를 실행하며 실습을 진행했습니다.

* **자료형의 종류 및 특징 실습**
  * **숫자, 문자열, 불(Boolean):** 데이터의 가장 기본적인 타입 이해
  * **변수, 리스트, 튜플, 딕셔너리, 집합:** 데이터를 효율적으로 담고 관리하는 자료구조의 특징 파악 및 활용

* **제어문 (Control Flow Statement) 실습**
  * **조건문 (`if` / `elif` / `else`)**: 
    * 비교 연산자 및 논리 연산자(`and`, `or`, `not`)의 `Bool` 자료형 반환 메커니즘 이해
    * 멤버십 연산자(`x in s`, `x not in s`)를 활용한 데이터 필터링 기초 조건 구성
    * 아무 일도 수행하지 않고 넘어갈 때 사용하는 `pass` 키워드 활용
    * 한 줄로 간결하게 조건을 처리하는 **조건부 표현식 (Conditional Expression)** 마스터
  * **반복문 (`while` / `for`)**:
    * 조건식이 `True`인 동안 무한히 반복되는 `while`문의 구조와 안정적인 탈출 조건 설계
    * `f-string` 포맷팅을 결합한 가독성 높은 반복 데이터 출력 다듬기
    * 루프의 흐름을 강제로 깨뜨리는 `break`와 다음 루프로 건너뛰는 `continue`의 동작 제어 차이 분석
    * `range()` 함수를 연동한 인덱스 기반 반복 및 다중 루프를 활용한 **구구단 연산** 구현
  * **리스트 내포 (List Comprehension)**:
    * 파이썬 고유의 강력한 문법인 리스트 내포의 기본 구조(`[표현식 for 항목 in 반복가능객체]`) 파악
    * 반복문 내부에 `if` 조건·분기 필터링을 결합하여 메모리 효율적이고 간결한 데이터 가공 및 리스트 생성 기법 체득
